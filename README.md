# AGNMLCC

This repository provides the machine-readable catalogs associated with the paper:

**A Systematic Search for Isolated Microlensing Candidates in ZTF AGN Light Curves**

The catalogs contain temporally isolated, single-peaked, approximately symmetric, PSPL-like flare candidates selected from Zwicky Transient Facility (ZTF) AGN light curves.

## Files

* `AGNMLCC.csv`
  Main AGN candidate catalog. This catalog contains 304 isolated PSPL-like flare candidates and is used for the main statistical analysis.

* `AGNMLCC_QZO_auxiliary_catalog.csv`
  Auxiliary QZO-only catalog. This catalog contains 77 candidates from the source-only QZO subset and is not included in the main event-rate or optical-depth estimates.

* `readme.txt`
  Detailed catalog description, column definitions, and notes on sample construction.

## Notes

These sources are microlensing candidates / PSPL-like flare candidates, not confirmed microlensing events.
